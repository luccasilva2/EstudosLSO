# **App Name**: StudyForge

## Core Features:

- Course Catalog: Display courses fetched from static JSON data, categorized by source (IFC/MEC).
- Expandable Accordions: Use accordions to show lists of files with animated expansion and collapse.
- File Viewer: Render uploaded study material PDFs in the browser, linking other file types for download.
- Animated Transitions: Apply animated page transitions powered by Framer Motion, including loading sequences.
- Responsive Navbar: Fixed top navbar with navigation links and a functional search bar that scans all files.
- Themed UI Elements: Apply code-themed styles with subtle code background patterns to enhance user experience.

## Style Guidelines:

- Primary color: Vivid orange (#FF5733), evoking energy and creativity, combined with technological vibes.
- Background: Dark gray (#333333) with desaturation at 20% for a clean tech aesthetic.
- Accent color: Electric blue (#7DF9FF), complementary to orange and inspired by code syntax highlighting.
- Headline font: 'Space Grotesk', sans-serif, providing a computerized feel suitable for headlines.
- Body text: 'Inter', sans-serif, providing a clean and modern feel for body text.
- Use React Icons such as FaCode, FaFilePdf to clearly represent file types and categories.
- Employ a grid-based layout for course cards, enhancing readability and visual structure.
- Use fade-in effects with Framer Motion for initial page loads and smooth transitions throughout the app.
